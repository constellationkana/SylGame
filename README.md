# SylGame

SylGame is a Unity 6 prototype with a complete playable loop: collect the required objectives, survive and fight the patrolling enemy, reach the exit door, and complete the level.

## Prototype Status

Current target: `v0.1.0`

The prototype includes:

- Player movement, sprinting, and jumping
- First-person and third-person cameras with runtime switching
- Interaction, collectible, objective, and exit-door systems
- HUD, game over, restart, and level complete UI
- Enemy patrol, detection, chase, contact damage, health, and death feedback
- Level blockout v01

## Unity Version

Open the project from `SYL/` with Unity `6000.4.10f1`.

## Controls

- Move: WASD
- Look: Mouse
- Sprint: Left Shift
- Jump: Space
- Attack: Left Mouse Button
- Interact: E
- Switch camera: V
- Restart after game over: R
